#  Event Management System

## Project Overview

 Event Management System is a web-based application developed using Flask and MySQL to manage events efficiently.

The system allows administrators to create and manage events while students can view, search, and register for events online.

## Features

### Admin Module

* Admin Login
* Add Event
* Edit Event
* Delete Event
* Manage Events
* Event Reports
* Dashboard Analytics

### Student Module

* Student Registration
* Student Login
* View Events
* Search Events
* Register for Events
* View Registered Events

## Technologies Used

* Python
* Flask
* MySQL
* HTML
* CSS
* JavaScript
* GitHub

## Database Tables

* students
* admins
* events
* registrations

## How to Run

1. Clone repository
2. Install dependencies
3. Configure MySQL database
4. Run app.py
